# Baisanat Landing

Корпоративный лендинг холдинга **Baisanat** на фреймворке [Astro](https://astro.build/).

## О проекте

- **Фреймворк:** Astro v6
- **Репозиторий:** https://github.com/mr-diamant/baisanat
- **Хостинг:** https://me-qr.kz/baisanat/
- **Дизайн:** [Figma (страница Baisanat)](https://www.figma.com/design/YENNDuHTEr1Pa4thlnprWi/Untitled?node-id=19-2&t=v7u1Yl2ts6n6gXbU-1)

## Структура

```text
src/
├── components/
│   ├── Header.astro       # Шапка: логотип, язык, меню
│   ├── Hero.astro         # Первый экран (hero)
│   ├── HeroCard.astro     # Карточка направления
│   ├── Ecosystem.astro    # Блок 02: Экосистема, карточки, подразделения и услуги
│   ├── Principles.astro   # Блок 03: Принципы
│   ├── Projects.astro     # Блок 04: Проекты
│   ├── Clients.astro      # Блок 05: Клиенты
│   ├── Cooperation.astro  # Блок 06: Сотрудничество
│   └── Footer.astro       # Подвал сайта
├── pages/
│   ├── index.astro        # Главная страница
│   ├── terms.astro        # Пользовательское соглашение и Политика конфиденциальности (светлая тема)
│   ├── privacy.astro      # Редирект на объединенную страницу /terms
│   └── contacts.astro     # Страница контактов (тёмная тема, интерактивная карта)
└── styles/
    └── global.css         # Глобальные стили и CSS-переменные

public/
├── Media/                 # Изображения, логотипы и JSON файлы тегов
├── logo.png               # Логотип Baisanat Holding
├── favicon.svg
└── favicon.ico
```

## Команды

| Команда           | Действие                                    |
| :---------------- | :------------------------------------------ |
| `npm install`     | Установить зависимости                      |
| `npm run dev`     | Запустить dev-сервер на `localhost:4321`    |
| `npm run build`   | Собрать статический сайт в `./dist/`        |
| `npm run preview` | Предпросмотр сборки локально                |

## Статус

Проект находится в процессе активной разработки. Свёрстаны и настроены основные блоки главной страницы: Шапка, Первый экран (Hero), Экосистема (с горизонтальным скроллом карточек, динамическими услугами и сервисами), Принципы работы, Проекты, Сетка клиентов, Форма сотрудничества и Подвал.

Также добавлена объединенная страница юридических документов:
- **Пользовательское соглашение и Политика конфиденциальности** (`/terms`) в светлом оформлении со стилизованным списком, адаптивным пошаговым отображением и эффектом пишущей машинки на заголовке.
- Реализован автоматический редирект с адреса `/privacy` на `/terms`.

## Деплой

Сборка (`./dist/`) выгружается на хостинг `me-qr.kz/baisanat/`.
